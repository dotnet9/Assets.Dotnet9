## 2. 准备

创建一个WPF工程，比如站长使用 [.NET 7](https://dotnet.microsoft.com/zh-cn/) 创建名为 **Login5** 的WPF项目。

![](https://lequ.co/2022/05/4702.png)

找一张图片做为装饰，放登录表单左侧：

![](https://lequ.co/2022/05/4703.png)

添加Nuget包 [MaterialDesignThemes](https://dotnet9.com/2020/12/Material-designinxaml-an-open-source-csharp-WPF-Control-Library)：


```xml
<PackageReference Include="MaterialDesignThemes" Version="4.6.0-ci176" />
```

## 3. 简单说明 + 源码

界面比较简单，代码也不多，我们直接贴代码。

**MainWindow.xaml**

界面的整体布局和样式都在这个文件内：

```xml
<Window

    xmlns:materialDesign="http://materialdesigninxaml.net/winfx/xaml/themes"
    xmlns:mc="http://schemas.openxmlformats.org/markup-compatibility/2006"
    Title="MainWindow"
    Width="800"
    Height="450"
    ResizeMode="NoResize"
    WindowStartupLocation="CenterScreen"
```

**MainWindow.xaml.cs**

窗体拖动和关闭按钮事件：

```C#
using System.Windows;
using System.Windows.Input;

namespace Login5;

public partial class MainWindow : Window
{
    public MainWindow()
    {
        InitializeComponent();
    }

    private void Border_MouseDown(object sender, MouseButtonEventArgs e)
    {
        this.DragMove();
    }

    private void btnExit_Click(object sender, RoutedEventArgs e)
    {
        this.Close();
    }
}
```
